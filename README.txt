Instalace
1. Nainstalujte MySQL server a vytvořte databázi za pomoci skriptu, který je ve složce sql s názvem database.sql
2. Přidejte adresu databázového server, uživatele a jeho heslo do konfiguračního souboru
3. Spusťte program pomocí příkazu python src/main.py
